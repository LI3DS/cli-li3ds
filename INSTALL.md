# micmac_li3ds installation

1. Clone the repo:

   ```bash
   $ git clone git@github.com:LI3DS/micmac_li3ds.git
   ```

1. Create a virtual env

   ```bash
   $ virtualenv li3ds
   ```

1. Install:

    ```bash
    (li3ds) $ cd micmac_li3ds
    (li3ds) $ pip install -e .
    ```

1. Use:

    ```
    (li3ds) $ mm2li --help
    ```
